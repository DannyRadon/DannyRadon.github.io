The Car Data CRUD (Create, Read, Update, Delete) Program.

It reads data from a CSV or Excel file in the format:   Make | Model | Year | Cost | Range | Comment

When loaded, you are greeted with the main menu giving you various options:

1.) Load Car Data - This Displays All Vehicle Entries in the CSV File
2.) Add a Car - This Allows the User to Add a Vehicle Entry to the CSV File
3.) Modify a Car - This Allows the User to Modify an already Exisiting Vehicle Entry in the CSV File
4.) Delete a Car - Allows the User to Delete a Vehicle Entry from the CSV File
5.) Save Data - Saves the Changes to the CSV File
6.) Report on Top 3 - Reports Back the Top 3 Economic Vehicles
7.) Quit - Terminates the Program
