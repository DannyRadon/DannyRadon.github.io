to play slots just call it up;
    python slots.py

to play blackjack you can call it up:
    python blackjack.py


*Playing Slots*

After loading slots, there will be a menu.
1.) Make a Bet - Begin Playing
2.) Exit Slots - Exit the Game

**Placing a Bet**
You will have a menu to select the amount you wish to bet. There is an option for a custom amount if the user desires. Once satisfied, select "Place Bet" to start the game and play.


*Playing BlackJack*

After loading blackjack, there will be a menu.
1.) Place a Bet - Begin Playing
2.) Ask for Spot - Under Construction (Ask for a Loan)
3.) Sitout Hand - Under Construction (Computer Plays Itself)
4.) Leave Table - Exit Blackjack

**Placing a Bet**
You will have a menu to select the amount you wish to bet. There is an option for a custom amount if the user desires. Once satisfied, select "Place Bet" to start the game and play.
