Dan & Dave's Dangerous Dig! (Tech Demo)

Requirements - PyGame (2.6.1)

**NOTES:**
Since this is an early version tech demo, I haven't yet implemented the needed code for the 'Dan' character but
you can play this tech demo as the 'Dave' character (black hair with beard)


**How to Use**
Open the Game by calling it:
    python play_dangerous_dig.py

**Game Information**

*Main Menu*
New Game - Begin a New Game
Load Game - Pick Up Where You Left Off
Options - Change Game, Video, Music Settings
Credits - Our credits
Exit Game - Close D&D's Dangerous Dig

*Difficulties*
Easy - For Beginners; more items, more lives, less enemies
Normal - The baseline it was designed around
Hard - Less items, less lives, more enemies

*Items/Pickups*
Gold Coin - Collect 100 to Earn an Extra-Life!
Pick Axe - A Throwable Weapon
Dynamite - A Throwable Explosive
Health Kit - Replenishes 3 Health
Bottled Water - Replenishes 1 Health
Key - Sometimes Needed to Pass the Level
